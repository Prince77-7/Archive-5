# Shelby County Parcel Viewer

An interactive web application that displays Shelby County parcels on a map and allows users to view detailed property and ownership information.

## Features

- Interactive map showing all parcels in Shelby County
- Click on parcels to view detailed ownership information
- Search functionality by address or owner name
- Property details including:
  - Parcel ID and address
  - Owner name and mailing address
  - Property assessment information
  - Zoning and land use details
- Mobile-responsive design

## Technologies Used

- [ArcGIS API for JavaScript 4.28](https://developers.arcgis.com/javascript/latest/)
- Shelby County GIS REST Services
- HTML5/CSS3/JavaScript

## Data Source

This application uses the Shelby County GIS REST services:
- Parcel data from `https://gis.shelbycountytn.gov/arcgis/rest/services/Parcel/CERT_Parcel/MapServer`

## How to Use

1. Open the `index.html` file in a web browser
2. Navigate the map to find parcels of interest
3. Click on a parcel to view detailed ownership information
4. Use the search box to find properties by address or owner name

## Running Locally

Simply open the `index.html` file in a modern web browser.

## Browser Compatibility

This application works best in modern browsers:
- Chrome
- Firefox
- Safari
- Edge
